=====Penjelasan Framework
--Model
Item		: model untuk membuat fungsi item yang akan digunakan (player, dan platform/batang)
ObjectIDS	: model untuk memberikan ID untuk setiap item
Platform	: model untuk membuat batang yang akan dipijak
Player		: model untuk player

--View
GameState	: untuk menentukan kondisi tampilan game
Menu		: untuk menampilkan menu awal sebelum masuk ke dalam game dalam bentuk form
Window		: untuk menampilkan tampilan window game 

--ViewModel
KeyHandler	: viewmodel untuk mengatur inputan gerakan player
LevelHandler	: viewmodel untuk mengatur jalannya game
